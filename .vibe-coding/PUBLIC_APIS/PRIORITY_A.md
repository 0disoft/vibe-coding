# A급 상세 문서 대상 목록

실수 한 번이 비용/사고로 직결되는 API들입니다.
이 목록의 Provider는 `providers/<id>/` 폴더에 상세 문서를 **반드시** 작성합니다.

## A급 기준

- 결제/구독/인보이스
- OAuth 기반 로그인/권한
- 메시지 발송 (SMS, 알림톡, 푸시)
- 보안 인텔리전스/위협 차단
- **개인정보 직접 처리** (계정 연동, 신원확인, 민감 데이터)

## 체크리스트 (문서 작성 시)

상세 문서 작성 시 반드시 포함할 항목:

- [ ] 인증 방식 및 환경변수명

- [ ] 웹훅 서명 검증 방법
