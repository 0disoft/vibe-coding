# 거짓 예언의 문장

| 항목 | 내용 |
|------|------|
| **표시명** | 거짓 예언의 문장 |
| **유형** | 단서 |

| **회수 예정** | 6~7화 |
| **상태** | 심기 |

## 훅 요약

- “마나폭주가 시작되면, 가장 약한 자가 열쇠가 된다”는 문장은 일부러 왜곡된 정보다.

## 심기(Seeding)

- 독자에게 보이는 단서: 문장이 유물(벽문/서판)에 남아 있고, 여러 인물이 서로 다르게 해석한다.
- 숨겨진 의도: ‘약한 자’는 전투력이 아니라 “저항력/적성”을 뜻한다.

## 강화(Reinforcement)

- ‘약한 자’를 전투력으로 착각한 선택이 작은 실패로 이어지게 만든다.
- 반대로 다른 해석의 가능성을 암시하는 장면(대사/관찰)을 한 번 넣는다.

## 회수(Payoff)

- 회수 방식: 마나폭주를 멈추는 열쇠가 “취약성”이 아니라 “특정 적성”임을 밝혀 오해를 뒤집는다.
- 회수 시 독자 감정: 반전 + “처음부터 단서가 있었네” 느낌을 주도록 심기 장면을 회상시킨다.

## 노트

- ‘거짓 예언’은 완전한 거짓이 아니라 “정의가 다른 진실”로 설계한다(회수 만족도↑).
