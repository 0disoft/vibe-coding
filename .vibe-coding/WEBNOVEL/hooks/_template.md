# 복선/떡밥(Hook) 템플릿

새 복선/떡밥을 만들 때 이 파일을 복사하여 `영문-슬러그.md` 로 저장하세요.

- 파일명: `영문-슬러그.md` (예: `mysterious-scar.md`)
- 표시명은 아래 테이블에 한글로 작성

---

| 항목 | 내용 |
|------|------|
| **표시명** | (한글 이름, 예: 수수께끼의 흉터) |
| **유형** | (미스터리/단서/약속/세계관 규칙/인물 비밀 등) |
| **심은 화** | (예: 3화) |
| **회수 예정** | (예: 12~15화, 미정 가능) |
| **상태** | (심기/강화/회수/보류/폐기) |
| **연관 요소** | (예: `characters/han-seo-jun.md`, `objects/ashen-sword.md`) |
| **태그** | (예: #미스터리 #정체 #회수필수) |

## 훅 요약

- 독자가 “뭐지?” 하고 기억해야 하는 한 문장

## 심기(Seeding)

- 심을 때 독자에게 보이는 단서(장면/대사/묘사)
- 숨겨진 의도(작가만 아는 의미)

## 강화(Reinforcement)

- 중간에 한두 번 더 떠올리게 만드는 장치(반복/변주/오해)

## 회수(Payoff)

- 회수 방식(반전/정체 공개/갈등 해결/희생 등)
- 회수 시 독자 감정(카타르시스/공포/허무/감동 등)

## 노트

- 일관성 체크 포인트(모순 방지)
- 대체 회수안(필요 시)
